# MovieLens-KG-
MVP简介：基于知识图谱的电影推荐系统**  本实验构建了一个结合**用户行为数据**和**电影元数据**（如类别、导演、演员等）的电影推荐系统MVP。我们使用**MovieLens**数据集和**IMDb**或**DBpedia**提取的电影元数据，通过**协同过滤**或**矩阵分解（SVD）**方法提升推荐效果。数据预处理包括生成**用户-电影评分矩阵**，并将电影的元数据转化为特征。模型评估使用**RMSE**、**Precision**、**Recall**等指标，实验表明，知识图谱的引入有效提高了推荐准确性。  该MVP为后续优化（如图神经网络）提供了基础，展示了知识图谱在推荐系统中的潜力。
